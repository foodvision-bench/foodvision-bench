# foodvision-bench

Reproducible benchmarks for food-image recognition models and commercial
nutrition APIs.

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

## What this is

`foodvision-bench` will be a small Python package that runs a reproducible
evaluation of food-image recognition systems against a held-out test set.
The goal is to give researchers, engineers, and reviewers a common,
inspectable yardstick: one repo, one test set, one set of metrics, one
leaderboard file updated every couple of months.

Everything is MIT-licensed; if you disagree with a number, open a PR with
a better measurement.
